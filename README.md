# Usage
```
JHSoundWaveConfig *config = [[JHSoundWaveConfig alloc] init];
config.count = 26;
JHSoundWaveView *view = [[JHSoundWaveView alloc] initWithFrame:CGRectMake(100, 100, 10, 20) config:config];
[self.view addSubview:view];
```
